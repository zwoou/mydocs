[TOC]

## 进程（process）

一个进程就是一个正在执行程序的实例，包括程序计数器、寄存器和变量的当前值。

### 进程的创建

4种主要事件会导致进程的创建

1. 系统初始化
2. 正在运行的程序执行了创建进程的系统调用
3. 用户请求创建一个新进程
4. 一个批处理作业的初

### 进程的终止

通常由下列条件引起

1. 正常退出（自愿的）
2. 出错退出（自愿的）
3. 严重错误（非自愿）
4. 被其他进程杀死（非自愿）

### 进程的实现

为了实现进程模型，操作系统维护着一张表，即进程表（process table）.每个进程占用一个进程表项。
